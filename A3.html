Abbreviations

It is used to mark up an abbreviation or acronym: IUPAC

Blockquote

It is used to present content from another source:

I am on a Mission to save a Magician

From Henry Danger

Multiple Code Lines

For multiple lines of codes use the pre element

            Codingal is 
            on a mission
            to inspire all the school 
            kids

Keyboard Input

To indicate input that is entered via keyboard you use the kbd element

ctrl + x is used to cut an item in a document
